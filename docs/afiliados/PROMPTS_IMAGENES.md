# Prompts para generar las imágenes (Stitch / Imagen / Midjourney / DALL·E)

Reglas comunes para TODOS: **sin texto dentro de la imagen** (el texto lo ponen los
banners), **fondo transparente** donde se indica, paleta de marca cian `#00EEFC`,
verde `#39FF14`, morado `#C464FF`, fondo `#0B1020`. Público: niños y adolescentes 8-18.

> Nota: Google Stitch está más orientado a interfaces. Si no genera bien ilustración/foto,
> usa estos mismos prompts en Imagen, Midjourney o DALL·E.

---

## 1. Mascota Lumi (PNG, fondo transparente)
```
Mascota robot amigable llamada "Lumi", guía de una app de inglés para niños y
adolescentes (8 a 18 años). Cuerpo redondeado y simpático, visor/ojos grandes
brillantes en cian, pequeña antena, expresión cálida y acogedora (nunca aterradora).
Cuerpo blanco brillante con acentos neón: cian #00EEFC, verde #39FF14 y morado #C464FF.
Suave resplandor neón. Cuerpo completo, vista frontal, centrado. Fondo TRANSPARENTE.
Estilo ilustración 3D limpia y moderna, alta calidad. SIN texto.
Generar 3 poses: saludando (mano arriba), celebrando (brazos arriba) y pensando.
Formato cuadrado 1:1, alta resolución.
```

## 2. Foto "Duelo Familiar" (lifestyle)
```
Foto realista y cálida de un padre/madre latino y su hijo (10-14 años) riendo y
compitiendo con un celular, pasándose el teléfono, en casa, luz natural, ambiente
acogedor, fondo levemente desenfocado. Emoción genuina, alegre y cómplice. Sin texto
en pantalla. Alta calidad, estilo candid (espontáneo). Entregar en vertical 9:16 y en
horizontal 1.91:1.
```

## 3. Fondo neón abstracto (para poner texto encima)
```
Fondo tecnológico oscuro para la marca de una app de inglés para niños. Base azul
muy oscuro casi negro (#0B1020), sutil grid/partículas neón, resplandores suaves en
cian (#00EEFC), verde (#39FF14) y morado (#C464FF), un motivo tenue de rayo. Moderno
y limpio, con mucho ESPACIO NEGATIVO al centro para poner texto encima. Sin personajes,
SIN texto. Entregar en 9:16, 1:1 y 1.91:1.
```

## 4. Niño celebrando (prueba social / testimonios)
```
Niño feliz (8-14 años) celebrando un logro, puño arriba, sosteniendo un celular,
chispas/destellos neón cian y verde alrededor, fondo transparente o azul oscuro simple.
Para gráficas de testimonio y prueba social. Estilo realista o 3D estilizado. SIN texto.
Formato 1:1.
```
