# PWA icons

Placeholder folder. Generate **192×192** and **512×512** PNGs of the IdiomaConnect logo and drop them here as:

- `icon-192.png`
- `icon-512.png`

Quick way: take any IdiomaConnect logo, run it through https://realfavicongenerator.net/ and dump the resulting `android-chrome-*.png` files in this folder renamed as above.

The maskable purpose in `manifest.webmanifest` expects ~20% safe-zone padding inside the canvas.
